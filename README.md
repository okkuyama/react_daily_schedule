# react_daily_schedule
React (redux) daily schedule calendar

## Initialize
```
$ yarn initial
```

## Run
```
$ yarn start
```

## Setting
### API URL
Path: `/src/common/config/constApi.js`
Change Value: `apiBaseUrl`

### MockAPI Change
Path: `/src/daily_schedule/apis/dailySchedule.js`
`adapter: mockAdapter` to comment out.
`params` to enable.
